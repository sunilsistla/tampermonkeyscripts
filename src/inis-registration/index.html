Introduction

This plugin helps you to auto fill the INIS appointment registration form.
You can setup multiple profiles and pick one to auto fill.

Setup

  1. This plugin runs on TamperMonkey. Install TamperMonkey Plugin on your browser. Restart browser after installation.
  2. Launch INIS registration helper and click install.
  3. Launch INIS Profiles and click install.
    NOTE: Refer Configuration to setup your profile. Re-installing this script will reset configuration. Save your existing profiles before re-installing.
  4. GoTo INIS Registration portal and click "Make appointment"to start.

Configuration

This script requires one time setup for every profile/ application.

  1. Click on Tamper Monkey Extension in your browser. Click on edit icon on "INIS Profiles" item.
    Follow Tamper Monkey tutorial and update profile information in INIS profiles script.
  2. Fill the fields as per the form in INIS Registration portal.
